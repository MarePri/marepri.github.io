- **CSS** — custom styles with Barlow font, glassmorphism UI, and animated background elements
- **JavaScript** — parallax effect via Parallax.js, hover state toggling
- **CDN dependencies**: Parallax.js 3.1, jQuery 2.1.3

## How to Run Locally

```bash
# Clone the repo
git clone https://github.com/MarePri/marepri.github.io.git

# Open directly in browser
cd marepri.github.io
start index.html

# Or use a local server (recommended)
npx serve .
```

Then open `http://localhost:3000` (or wherever `serve` tells you).
